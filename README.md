# 🏡 Дом-трекер под Варшавой

Интерактивная карта и карточки объектов (дома/близняки, первичка) в зонах Lesznowola · Magdalenka · Słomin · Raszyn · Nadarzyn (+ резерв). Бюджет ≤900k PLN без отделки.

**Открыть:** `index.html` (двойной клик) или через GitHub Pages.

- 🗺 Карта (OpenStreetMap + Leaflet) со всеми объектами
- 🎴 Карточки с фильтром по статусу, ценами и ссылками на застройщиков
- 🔗 Кнопки: сайт застройщика · портал/фото · Google Maps · «на карте»

Данные: сайты застройщиков и порталы (rynekpierwotny, otodom, tabelaofert, nieruchomosci-online). Обновлено 30.05.2026. Не является офертой.

## Деплой на GitHub Pages
1. Создать пустой репозиторий на GitHub (например `dom-tracker`).
2. `git remote add origin git@github.com:ЛОГИН/dom-tracker.git`
3. `git push -u origin main`
4. Settings → Pages → Branch: `main` → `/(root)` → Save.
5. Адрес: `https://ЛОГИН.github.io/dom-tracker/`
